## Whatâ€™s Inside The Tin

The following functions are implemented:

  - `card_widget`: Turn an htmlwidget into a web deployable, interactive
    Twitter card
  - `gg_preview`: Generate a Twitter Player card preview image from a
    ggplot2 plot
